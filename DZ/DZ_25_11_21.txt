1. Требуется написать скрипт для vba.xlsm (lesson 25) для листа с именем "dz"
2. Скрипт должен вывести индексы последней заполненой строки и последнего столбца
3. После п.п. 2 скрипт должен создать новое поле с названием "тест" в ячейке "D1"
4. После п.п. 3 написать цикл по таблице с условием:
    4.1 если в поле "idtest" будет "ошибка" И в поле "text_test" тоже будет "ошибка",
     то нужно в поле "тест" в этой строке написать индекс строки + "errors" + окрасить цвет текста ячейки в красный (посмотрите номер инекдс цвета в интернете)
    4.1 и если в поле "idtest" будет "ошибка" ИЛИ в поле "text_test" будет "ошибка",
     то нужно в поле "тест" в этой строке написать индекс строки + "error" + окрасить цвет текста ячейки в желтый (посмотрите номер инекдс цвета в интернете)
    4.3 во всех остальных случаях в поле "тест" в этой строке написать индекс строки + "no error" + окрасить цвет текста ячейки в зеленый (посмотрите номер инекдс цвета в интернете)
