Сложное задание
Пример можно посмотреть на листе "example"
1. Требуется написать скрипт для test(график).xlsx (lesson 26) для листа с именем "test"
2. Скрипт должен создать название поля "difference" в J1
3. После создания названия поля нужно заполнить ячейки в столбце J:
    3.1 В рамках одного массива (number_list) и типа алгоритма(alg_type) смотрим текущего автора алоритма(author)
        и ищем разницу в столбце "count_oper" относительно других авторов (author)
    3.2 Если "count_oper" отличается, то в J добавляем информацию автора.Если их несколько, то перечислям через запятую.
    3.3 Если "count_oper" не отличается, то оставлям J пустым
4. Обрати внимание, что есть selection_sort и selection_sort_2 в alg_type,это один алгоритм!!!

Легкое задание
1. Требуется написать скрипт для test(график).xlsx (lesson 26) для листа с именем "test"
2. Скрипт должен создать название поля "join" в K1
3. После создания названия поля нужно заполнить ячейки в столбце K:
    3.1 Требуется объединить иноформацию всех полей number_list,count_oper,alg_type,author,res_len,res_min,res_max,res_unique,time_alg
    через запятую в ячейке стобца K
    3.2 Условием объединения служит поле "time_alg". Оно должно быть больше 0.00000000000
    3.3 Если поле равно или меньше 0.00000000000, то ячейка стобца K пустая
